# RPG Platform 
Projeto de uma plataforma de RPG estilo tabletop (inspirado em Roll20).

## Tecnologias

* React + Vite
* TypeScript
* Zustand

## Instalação

Clone o projeto:

```bash
git clone https://github.com/seu-usuario/rpg-platform.git
```

Instale as dependências:

```bash
npm install
```

## Rodar o projeto

```bash
npm run dev
```

Acesse:

```
http://localhost:5173
```
